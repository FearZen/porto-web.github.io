Pembuatan Website Risenmedic

Website Risenmedic adalah platform yang dirancang untuk memudahkan pengguna dalam berbagai aspek perawatan kesehatan. Dengan menggunakan MySQL sebagai database, website ini memberikan layanan untuk:

  • Membeli obat secara online.
  • Mencari informasi obat dan penyakit.
  • Mencari lokasi klinik atau apotek terdekat.

Website ini juga mengintegrasikan API Google Maps untuk membantu pengguna menemukan lokasi klinik terdekat dengan mudah.
